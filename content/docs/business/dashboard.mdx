# Dashboard Overview

Your dashboard is your central hub for managing projects, applications, and student interactions.

## Dashboard Layout

### Main Sections

**Top Navigation:**
- Projects
- Applications
- Messages
- Analytics
- Settings

**Overview Widget:**
- Active projects count
- Pending applications
- Messages unread
- Quick actions

## Projects Section

### Active Projects

View all your posted projects:

**Project Cards Show:**
- Project title and category
- Status (Open, In Progress, Completed)
- Number of applications
- Posted date
- Application deadline

**Quick Actions:**
- Edit project details
- View applications
- Close applications
- Mark as completed
- Archive project

### Project Status

**Open:**
- Accepting applications
- Visible to students
- Can receive messages

**In Progress:**
- Student selected
- Work underway
- Tracking milestones

**Completed:**
- Project finished
- Archived for records
- Can provide endorsement

## Applications Section

### Application Management

**Filter Options:**
- By project
- By status (pending, accepted, rejected)
- By date received
- By student rating

**Bulk Actions:**
- Send message to multiple applicants
- Update status
- Export applications

### Application Details

**Student Information:**
- Profile overview
- Skills match percentage
- Availability
- Portfolio links

**Application Content:**
- Cover letter
- Responses to questions
- Submitted materials
- Application date

## Messages Section

### Chat Interface

**Conversation List:**
- Active conversations
- Unread message indicators
- Last message preview
- Student name and project

**Message Thread:**
- Full conversation history
- Send new messages
- Share files
- Mark important messages

## Analytics Section

### Project Performance

**Metrics Available:**
- Total projects posted
- Applications received per project
- Average time to fill projects
- Completion rate
- Student satisfaction ratings

**Application Analytics:**
- Application volume trends
- Source of applicants
- Skill match rates
- Response times

### Organization Stats

- Total students worked with
- Projects completed
- Average project duration
- Organization rating

## Settings Section

### Profile Settings

- Edit organization profile
- Update logo and branding
- Manage contact information
- Set privacy preferences

### Notification Preferences

**Email Notifications:**
- New applications
- Messages received
- Project updates
- Milestone completions

**In-App Notifications:**
- Real-time alerts
- Notification sound
- Desktop notifications

### Team Management

**Add Team Members:**
- Invite colleagues
- Set permissions
- Assign projects
- Manage access

## Quick Actions

### Frequently Used Features

**From Dashboard:**
- Post new project (one click)
- View pending applications
- Respond to messages
- Check analytics

**Keyboard Shortcuts:**
- `N` - New project
- `M` - Messages
- `A` - Applications
- `/` - Search

## Mobile Dashboard

### Mobile App Features

- View projects on-the-go
- Respond to applications
- Chat with students
- Approve milestones
- Receive push notifications
