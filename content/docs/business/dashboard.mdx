# Dashboard Overview

Your organization dashboard serves as the central hub for managing projects, reviewing applications, communicating with students, and tracking organizational performance on SkillBridge.

## Dashboard Layout

### Navigation Structure

**Primary Navigation Sections:**
Projects - View and manage all posted opportunities
Applications - Review and respond to student submissions
Messages - Direct communication with students
Analytics - Performance metrics and insights
Settings - Account and notification preferences

**Overview Dashboard Widget:**
Active projects count and status summary, pending applications requiring review, unread message notifications, quick action buttons for common tasks

## Projects Management Section

### Active Projects Display

Your projects section displays all opportunities you've posted with comprehensive status information.

**Project Card Information:**
- Complete project title and selected category
- Current status (Open, In Progress, Completed, Closed)
- Number of applications received
- Original posting date
- Application deadline (if set)

**Available Quick Actions:**
Edit project details and requirements, view all received applications, pause or close applications, mark project as completed, archive completed projects for records

### Project Status Indicators

<Tabs items={["Open Projects", "In Progress", "Completed"]}>
<Tab value="Open Projects">

**Open Status Characteristics:**
- Actively accepting new student applications
- Fully visible to students in browse and search
- Can receive direct messages from interested students
- Appears in relevant category filters

**Management Options:**
Edit project details, review incoming applications, pause acceptance temporarily, close applications permanently

</Tab>

<Tab value="In Progress">

**In Progress Status Characteristics:**
- Student selected and collaboration active
- Work currently underway on deliverables
- Milestone tracking enabled
- Regular communication and feedback exchanges

**Management Tools:**
Track milestone completion, provide feedback on submissions, communicate with student collaborator, request status updates, approve completed work

</Tab>

<Tab value="Completed">

**Completed Status Characteristics:**
- All project work finished and approved
- Automatically archived for organizational records
- Available for reference and review
- Opportunity to provide student endorsement

**Post-Completion Actions:**
Provide endorsement or recommendation, request testimonial for organization, document project outcomes, mark final compensation as paid

</Tab>
</Tabs>


## Applications Management Section

### Application Organization

**Filtering Capabilities:**
- Filter by specific project
- Sort by application status (pending, accepted, declined)
- Organize by submission date
- View by student match rating

**Bulk Action Tools:**
- Send messages to multiple applicants simultaneously
- Update status for multiple applications
- Export application data for external review
- Archive old applications

### Application Detail View

**Comprehensive Student Information:**
- Complete profile overview and background
- Skills match percentage based on requirements
- Stated availability and time commitment
- Portfolio links and work samples

**Application-Specific Content:**
- Personalized cover letter or message
- Responses to custom screening questions
- Any submitted supplementary materials
- Original application submission date

<Callout title="Application Review Efficiency" type="info">
Use the skills match percentage and filter tools to quickly identify the most qualified candidates, allowing you to focus detailed review time on top prospects.
</Callout>

## Messaging and Communication

### Integrated Chat Interface

**Conversation Management:**
- Active conversation list with all student communications
- Clear unread message indicators and counts
- Preview of last message in each conversation
- Associated student name and related project

**Message Thread Features:**
- Complete conversation history for context
- Rich text message composition
- File sharing capabilities for documents and resources
- Ability to star or mark important messages for reference

## Analytics and Insights

### Project Performance Metrics

**Available Data Points:**
- Total number of projects posted over time
- Average applications received per project
- Time-to-fill metrics for positions
- Project completion rate statistics
- Student satisfaction ratings and feedback

**Application Analytics:**
- Application volume trends over time
- Geographic source of applicants
- Skills match rates and quality
- Your average response time to applications

### Organizational Statistics

**Aggregate Performance Data:**
- Total unique students collaborated with
- Cumulative projects successfully completed
- Average project duration from start to finish
- Overall organization rating and reputation score

## Settings and Configuration

### Profile Management

**Organization Profile:**
- Edit comprehensive organization description
- Update logo, branding, and visual identity
- Manage primary contact information
- Adjust privacy and visibility preferences

### Notification Preferences

<Tabs items={["Email Notifications", "In-App Notifications"]}>
<Tab value="Email Notifications">

**Configurable Email Alerts:**
- New application submissions to your projects
- Direct messages received from students
- Project status updates and milestones
- Milestone completion notifications

**Frequency Options:**
Immediate (as they occur), Daily digest summary, Weekly summary, Disabled for specific categories

</Tab>

<Tab value="In-App Notifications">

**Platform Notification Settings:**
- Real-time alerts for time-sensitive updates
- Notification sound preferences
- Desktop browser notifications
- Mobile push notifications

**Customization:**
Enable/disable by notification type, Set priority levels, Configure quiet hours

</Tab>
</Tabs>

### Team Access Management

**Collaborative Features:**
- Invite team members or colleagues to platform
- Assign role-based permissions and access levels
- Delegate specific projects to team members
- Manage and revoke access as needed

## Dashboard Productivity Features

### Quick Access Tools

**One-Click Actions Available:**
- Post new project opportunity
- Review all pending applications
- Respond to unread messages
- Check recent analytics

**Keyboard Shortcuts** (for desktop):
- `N` - Create new project
- `M` - Jump to messages
- `A` - View applications
- `/` - Search functionality

### Mobile Dashboard Access

**Mobile App Capabilities:**
- View active projects on mobile devices
- Review and respond to applications on-the-go
- Chat with students via mobile interface
- Approve milestone submissions remotely
- Receive push notifications for important updates
