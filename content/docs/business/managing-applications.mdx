## Response Best Practices

### Timing
- **Ideal**: Respond within 24-48 hours
- **Maximum**: 1 week from application
- **Why**: Shows respect, maintains student interest

### Communication

**For All Applicants:**
1. Acknowledge receipt promptly
2. Set expectations for timeline
3. Be professional and respectful

**Accepted Applicants:**
```
Subject: Excited to work with you on [Project Name]!

Hi [Name],

[Your Name]
```

## Managing Multiple Applications

### Volume Guidelines

**Ideal Number:** 5-15 applications
- Enough to choose from
- Not overwhelming to review
- Good quality-to-quantity ratio

**If You Receive Too Many:**
1. Close applications early
2. Screen based on must-have requirements
3. Respond to all, even if brief

**If You Receive Too Few:**
1. Review project posting clarity
2. Consider expanding requirements
3. Extend application deadline
4. Promote project more actively

### Selection Process

**Step 1: Initial Screen (15 minutes)**
- Review all applications quickly
- Create shortlist of qualified candidates
- Eliminate clear mismatches

**Step 2: Detailed Review (30 minutes)**
- Deep dive into shortlist profiles
- Review portfolios thoroughly
- Compare qualifications

**Step 3: Communication (varies)**
- Ask clarifying questions
- Request additional materials if needed
- Schedule brief interviews (optional)

**Step 4: Final Selection (15 minutes)**
- Choose best fit
- Have backup choice
- Prepare to onboard

## Troubleshooting

**No Qualified Applicants?**
- Review skill requirements (too specific?)
- Check compensation (if competitive)
- Clarify project description
- Extend deadline
- Reach out to specific students

**Too Many Applicants?**
- Close applications
- Add screening questions
- Make requirements more specific
- Respond to shortlist first

**Student Non-Responsive After Acceptance?**
- Send follow-up message
- Provide clear next steps
- Set deadline for response
- Move to backup candidate if needed
