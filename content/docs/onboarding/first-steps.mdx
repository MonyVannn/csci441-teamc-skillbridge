# First Steps Guide

Welcome to SkillBridge! This guide will walk you through your first actions after creating an account.

## The First 24 Hours

### For Students: Quick Start Path

Your roadmap to getting started effectively:

#### â° First 30 Minutes
**1. Verify Your Email** (5 minutes)
- Check your inbox for verification email
- Click the verification link
- Confirm your email is verified

**2. Add Profile Photo** (5 minutes)
- Navigate to Profile settings
- Upload a professional photo
- Save your changes

**3. Write Your Bio** (15 minutes)
- Complete the "About Me" section
- 2-3 sentences about yourself
- Mention what you're studying and seeking

**4. Add Key Skills** (5 minutes)
- List your top 5-8 skills
- Set proficiency levels honestly
- Include both technical and soft skills

#### â° First 2 Hours
**5. Complete Education Section** (20 minutes)
- Add your current school/university
- Include your major and expected graduation
- List relevant coursework

**6. Add Experience (if any)** (30 minutes)
- Include internships, jobs, or volunteer work
- Add relevant school projects
- Focus on measurable achievements

**7. Browse Projects** (30 minutes)
- Visit the Browse page
- Explore different categories
- Read project descriptions
- Save interesting projects

**8. Set Your Preferences** (10 minutes)
- Set availability hours per week
- Configure notification settings
- Set communication preferences

#### â° Within 24 Hours
**9. Optimize Your Profile** (30 minutes)
- Add portfolio links (GitHub, LinkedIn, etc.)
- Review profile for completeness
- Fix any typos or errors
- Get to 80%+ completion

**10. Plan Your Applications** (30 minutes)
- Identify 3-5 projects that match your skills
- Read requirements carefully
- Prepare application materials
- Start with projects at your skill level

### For Businesses: Quick Start Path

Your roadmap to posting your first project:

#### â° First 30 Minutes
**1. Verify Your Email** (5 minutes)
- Check inbox for verification email
- Click verification link
- Confirm email is verified

**2. Upload Organization Logo** (5 minutes)
- Navigate to Organization Profile
- Upload your company logo
- Ensure it's clear and professional

**3. Write About Your Organization** (15 minutes)
- Draft your "About Us" section
- Include mission and what you do
- Mention why students should work with you

**4. Add Contact Information** (5 minutes)
- Provide primary contact email
- Add contact person name and title
- Include phone number (optional)

#### â° First 2 Hours
**5. Complete Organization Profile** (30 minutes)
- Select industry and organization size
- Add mission and values
- Include website and social links
- Add office location

**6. Research Best Practices** (30 minutes)
- Read the [Posting Projects guide](/docs/business/posting-projects)
- Review example project postings
- Understand what makes a good project
- Learn about project scope

**7. Define Your First Project** (45 minutes)
- Identify a well-scoped project
- Define clear deliverables
- List required skills
- Set realistic timeline

**8. Set Up Communication** (15 minutes)
- Test chat functionality
- Set email notification preferences
- Ensure primary contact can respond quickly

#### â° Within 24 Hours
**9. Post Your First Project** (45 minutes)
- Create detailed project posting
- Review for clarity and completeness
- Set application deadline
- Publish project

**10. Prepare for Applications** (30 minutes)
- Review student evaluation criteria
- Prepare interview questions
- Set up onboarding materials
- Plan project kickoff process

## Essential Actions

### Everyone Should Do

#### 1. Complete Profile to 80%+
**Why it matters:**
- Increases trust and credibility
- Better matching with opportunities
- Higher response rates
- Professional first impression

**How to achieve:**
- Fill all required fields
- Add recommended sections
- Include photo/logo
- Write compelling descriptions

#### 2. Set Communication Preferences
**Configure:**
- Email notifications (applications, messages, updates)
- Chat availability
- Preferred contact method
- Response time expectations

**Best practices:**
- Enable important notifications
- Disable noise/spam alerts
- Check SkillBridge at least once daily
- Respond within 24-48 hours

#### 3. Understand the Platform
**Take time to:**
- Read relevant documentation
- Explore the dashboard
- Test the chat feature
- Review FAQ for common questions

#### 4. Review Policies
**Important to understand:**
- Terms of Service
- Community Guidelines
- Privacy Policy
- Project completion policies
- Verification system
