# Finding Projects

Finding the right projects is key to building your portfolio and gaining valuable experience.

## Browse Page

### Accessing Projects

Navigate to **Browse** from the main menu to see all available projects.

**Default View:**
- Projects sorted by date posted (newest first)
- Category filters on left sidebar
- Search bar at top
- Quick view cards for each project

### Project Cards

Each card displays:
- Project title
- Organization name and logo
- Category and required skills
- Duration and time commitment
- Posted date
- Compensation (if any)
- Application status

## Project Details Page

### What to Review

Click on any project card to see full details:

**Project Information:**
- Complete description
- Specific deliverables
- Required and preferred skills
- Timeline and milestones
- Compensation details
- Application requirements

**Organization Profile:**

### Questions to Consider

Before applying, understand:

1. **What exactly needs to be done?**
   - Are deliverables clear?
   - Do I understand the scope?

2. **What support will I receive?**
   - Is mentorship mentioned?
   - Will someone guide me?

3. **What will I gain?**
   - Portfolio piece?
   - New skills?
   - Professional connections?

4. **What are the expectations?**
   - Quality standards?
   - Communication frequency?
   - Review process?

## Application Strategy

### How Many Projects?

**Beginners (First Project):**
- Apply to 3-5 well-matched projects
- Focus on quality applications
- Start with smaller scope projects
- Build confidence first

**Experienced (Have Completed Projects):**
- Apply to 2-3 at a time
- Can handle larger scope
- Mix challenging and comfortable
- Strategic portfolio building

> **âš ï¸ Warning**: Don't apply to too many at once! Quality over quantity.

### Timing Your Applications

**Best Times to Apply:**
- Within first 3 days of posting
- After thoroughly reviewing project
- When you have time to customize application
- When your schedule truly allows

**Avoid:**
- Applying minutes after posting (shows you didn't read carefully)
- Last-minute applications (appears desperate)
- Applying when overcommitted
- Hasty applications without preparation

## Search Tips & Tricks

### Finding Hidden Gems

**Look for:**
- Recently posted projects (less competition)
- Smaller organizations (more mentorship)
- Clear, detailed postings (better organizations)
- Mission-driven nonprofits (meaningful work)

### Advanced Search

**Combine Filters:**
```
Example: "React" + Web Development + Remote + 10-15 hrs/week
Example: "Marketing" + Nonprofit + Paid + Flexible timeline
Example: "Python" + Data Science + Beginner-friendly
```

### Set Up Alerts

**Get Notified:**
- Enable email alerts for new projects
- Set criteria for notifications
- Receive daily or weekly digests
- Never miss matching opportunities

## Understanding Competition

### Application Volume

Projects typically receive:
- **High-demand skills**: 15-30 applications
- **Specialized skills**: 5-15 applications
- **Niche projects**: 2-10 applications

**Your Advantage:**
- Complete profile
- Customized application
- Quick response time
- Relevant experience

### Standing Out

**Differentiate Yourself:**
- Apply early (but thoughtfully)
- Thoroughly customized application
- Strong portfolio examples
- Professional communication
- Clear enthusiasm and fit
